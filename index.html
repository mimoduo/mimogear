Mimogear

Static Site Generator

Mimogear compiles, distributes, watches, & deploys a pug template boilerplate, a postcss skeleton stylesheet, and a javascript module bundle with gulp & travis ci.

View on Github

Whip out a terminal and tryout Mimogear:

$ git clone https://github.com/mimoduo/mimogear.git
$ cd Mimogear
$ npm install
$ npm install gulp-cli -g
$ gulp
